# Coding Projects

This repository contains game development projects.

## Current Project: Color Ball Puzzle Game

A standalone HTML/JavaScript puzzle game that combines Tetris-like mechanics with color-matching gameplay.

### Game Features
- 8 different piece shapes composed of colored balls
- Match 3+ balls horizontally, vertically, or diagonally
- Gravity and cascade mechanics
- Progressive difficulty levels
- Score tracking and level progression
